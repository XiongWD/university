Result:

The crawler found many CHSI public pages and one Henan Education Examination Authority score-rank candidate page, but did not find a public full-volume Henan 2026 admissions catalog PDF/page.

### 2. Download and parse CHSI public pages

Downloaded 327 public CHSI pages. The extractor treated `.html`, `.htm`, `.dhtml`, and `.jsp` as HTML, then attempted conservative table extraction.

Result:

No parseable Henan 2026 university-major-group plan table was found. `normalized_catalog.csv` contains only the header.

### 3. Search-engine discovery

Searches were run for official-domain queries such as:

### 4. Henan Education Examination Authority probes

The discovered official page was:

```text
https://www.haeea.cn/a/202606/43717_14a087d2.shtml
```

It is the 2026 Henan score-rank statistics page, not the admissions catalog.

Adjacent article-id probing around the discovered page did not find `招生专业目录`, `招生计划`, `志愿填报指南`, or `院校专业组` pages.

Direct script access to `haeea.cn` had two blockers:

- TLS certificate chain validation failure in the local Python environment.
- When bypassing local TLS validation for public-page probing, the site returned HTTP `412 Precondition Failed`.

Do not bypass login, captcha, paywall, or access controls. If a browser can download the official PDF manually, place it under `data/raw/henan_2026/sources/` and rerun the extractor.
